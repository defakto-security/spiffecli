## spiffecli verify x509-svid

Verify an x509 SVID

```
spiffecli verify x509-svid [flags]
```

### Options

```
      --filename string                 Name of the file to read the x509 SVID from
      --format string                   Certificate format. Available options are pem or der
  -h, --help                            help for x509-svid
      --password string                 Certificate password, for password-protected DER files
  -s, --spiffe-endpoint-socket string   Path to Workload API socket (env: SPIFFE_ENDPOINT_SOCKET)
```
