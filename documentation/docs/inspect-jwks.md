## spiffecli inspect jwks

Inspect a JWKS (JSON Web Key Set)

```
spiffecli inspect jwks [flags]
```

### Options

```
      --color             Enable colorized output
      --format string     Output format is one of "json", "yaml", "summary", or "key-ids". (default "json")
  -h, --help              help for jwks
      --indent            Indent JSON output. Has no effect for other output formats
      --location string   URL or filename of bundle to inspect
```
