## spiffecli inspect jwt

Inspect a JWT

```
spiffecli inspect jwt [flags]
```

### Options

```
      --color             Enable colorized output
      --filename string   Name of input file containing the JWT
      --format string     Output format is one of "json", "yaml", or "summary". (default "json")
      --headers           Output header information
  -h, --help              help for jwt
      --indent            Indent JSON output. Has no effect for other output formats
      --isSvid            Return 0 if true, 1 if false. Disables other output options
      --timezone string   Timezone to use for "summary" format. Defaults to local timezone
```
