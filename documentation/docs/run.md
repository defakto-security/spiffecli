## spiffecli run

Runs a dev server that exposes a simulated Workload API

```
spiffecli run [flags]
```

### Options

```
      --config string   The dev configuration file, will be created if it doesn't exist (default "~/.spirl/dev.toml")
  -h, --help            help for run
```
