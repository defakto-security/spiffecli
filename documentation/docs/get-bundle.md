## spiffecli get bundle

Get a SPIFFE Trust Bundle

```
spiffecli get bundle [flags]
```

### Options

```
      --filename string       Name of the file to output results
      --format string         Format for the bundle (either x509 or jwt)
  -h, --help                  help for bundle
  -t, --trust-domain string   The bundle's trust domain
```

### Options inherited from parent commands

```
  -s, --spiffe-endpoint-socket string   Path to Workload API socket
```
